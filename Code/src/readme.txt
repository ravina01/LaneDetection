Name : Ravina M. Lad
Project Name : Advanced Lane Detection
Description :
To develop a software pipeline to identify the lane boundaries in a video from a front-facing camera on a car. Detect lane lines in a variety of conditions, including
changing road surfaces, curved roads, and variable lighting.

Python 3 and above, OpenCV 3 or above, NumPy, Matplotlib and MoviePy
